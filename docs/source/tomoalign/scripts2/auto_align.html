Script auto_align.py¶

Script to perform an automatic camera alignment (tilt & center).

The alignment typically needs a few iterations to obtain a perfect alignment.

usage: auto_align.py [-h] [-i MAXITERATIONS] [-r ROTATIONACCURACY]
                     [-c CENTERACCURACY] [-a REFERENCEANGLE]
                     [-f FLATFIELDPOSITION] [-p] [-s]

Named Arguments¶

-i, --maxiterations

The maximum number of iterations before giving up on achieving the specified alignment accuracy. (default = 6)

-r, --rotationaccuracy

The accuracy to which the camera rotation is to be aligned. The value specifies the maximum tilt measured in units of pixels over the larger of the two detector dimensions. (default = 0.25)

-c, --centeraccuracy

The accuracy, in units of pixels, to which the centering of the rotation axis with respect to the detector center is to be aligned. (default = 0.5)

-a, --referenceangle

The reference angle for the alignement images [degrees]. The first image will be taken at this angle, the second one 180 degrees from there. (default = 0.0)

-f, --flatfieldPosition

The position at which the flat field image is taken. If set to 0.0, the position will be calculated automatically.(default = 0.0)

-p, --donotplot

If the argument is given, the plotting of the individual alignment steps is suppressed. (by default plots are shown)

-s, --donotsave

If the argument is given, the result file and alignment plots are not saved to disk (by default results are saved)

EXAMPLES:

  • Perform a completely automatic alignment:

    auto_align.py

  • Perform the alignment to a specified minimum accuracy (0.8 pixels for the rotation and 1.0 pixels for the centering):

    auto_align.py -r 0.8 -c 1.0

  • Limit the maximum number of iterations to 8:

    auto_align.py -i 8
